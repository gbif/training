= Course description

[NOTE.description]
====
This course includes instruction to ensure learners are at the same level before delving into biodiversity data mobilization topics.

Topics include:

* Terminology and definitions of basic concepts, functions, and processes
* Introduction to data quality
* Importance of documentation

This course is comprised of video and written instruction.
====

== Audience

[NOTE.trainers]
====
This course is designed for anyone who is interested in completing the Biodiversity Data Mobilization course and sharing data with GBIF.
====

== Prerequisites

[NOTE.prep]
====
There are no prerequisites for this course.
This course serves as a prerequisite for the Biodiversity Data Mobilization course.
====

NOTE: While efforts are made to provide materials in other languages, videos will be in English. Efforts are made to include subtitles when possible.
