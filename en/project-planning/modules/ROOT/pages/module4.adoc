* Dataset Title 
* Dataset Type

* Geographic Scope 
* Temporal Scope
* Taxonomic Scope 
* Sampling Methodology (if relevant)
* Relevance [why this dataset?]
* Data Holders
* Has the data holder confirmed agreement to publish the data through GBIF?
* Expected Publication date
